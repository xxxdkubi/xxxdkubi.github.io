## 终端

### Fish Shell

比 zsh 更友好，开箱即用。自动补全、语法高亮，不需要折腾配置。

```bash
# 安装
brew install fish
```

### tmux

分屏神器，一个终端同时干多件事。SSH 断线也不怕。

## 编辑器：VS Code

用了几年，插件生态无出其右。

## 笔记工具：Obsidian

Markdown 笔记，本地存储，双向链接。

## 浏览器：Chrome + Arc

Chrome 用于开发调试，Arc 用于日常浏览。

## 结语

工具不在多，在于用熟。与其装 20 个插件，不如把 5 个核心工具用到极致。
